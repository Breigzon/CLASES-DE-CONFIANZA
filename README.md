# 🎓 TrustPreU - Clases de Confianza

**Solucionando la informalidad educativa en Perú mediante la red Stellar.**

### 🚀 El Problema
En Perú, las clases particulares se pagan por Yape sin garantías. Los alumnos temen estafas y los profesores jóvenes no tienen cómo demostrar su experiencia.

### 💡 Nuestra Solución
TrustPreU es un protocolo de confianza que usa Stellar para:
1. **Reputación Incorruptible:** Cada pago exitoso genera un historial de confianza real para el profesor en la blockchain.
2. **Descuentos Grupales:** Usamos lógica programada para que el precio baje automáticamente si más alumnos se inscriben, devolviendo la diferencia al instante.

### 🛠️ Stack Tecnológico
- **Blockchain:** Stellar (Testnet)
- **Smart Logic:** Stellar SDK & Soroban
- **Wallet:** Freighter
- **Frontend:** React + TypeScript + Tailwind CSS
